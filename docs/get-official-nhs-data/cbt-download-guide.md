# CBT download guide

Open the official NHS England
[*Cloud Based Telephony Data in General Practice* release index](https://digital.nhs.uk/data-and-information/publications/statistical/cloud-based-telephony-data-in-general-practice).
Choose a monthly release, then use its **Resources** section. For practice-level
activity obtain **Cloud Based Telephony By Day and Time**, participation and
submission evidence, metadata and supporting information. Obtain **Cloud Based
Telephony By Durations** when outcome or duration measures are required.

The [May 2026 CBT release page](https://digital.nhs.uk/data-and-information/publications/statistical/cloud-based-telephony-data-in-general-practice/may-2026)
provides the May resources and an official link to historic practice-level
data. Use the fixed manifests to identify the exact publication owner for each
earlier month rather than treating the May file as the source of every month.

CBT is official statistics in development with phased supplier coverage.
Accounts serving multiple practices can be unassigned because calls cannot be
reliably allocated to one practice. Treat invalid mappings, non-submission and
integrity gaps as missing evidence rather than zero activity.

Inspect current publication notices and historic-practice-level corrections,
including time-label changes and regional component warnings, before choosing a
release vintage.
