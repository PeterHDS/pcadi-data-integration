# OCS download guide

Open the official *Submissions via Online Consultation Systems in General
Practice* series. For practice-month integration obtain the practice-level
CSV/ZIP and metadata. Obtain the day-and-time CSV/ZIP and its metadata only when
performing temporal analysis.

Prefer the most recent reliable publication vintage that contains the required
historical months, but inspect the publication notices for incomplete supplier
submissions or retrospective changes. Record the selected vintage separately
from the analytical observation period.

Validate the practice code, month, supplier, metric and value fields. Registered
patients must be month matched. Supplier non-submission and non-participation
must not be converted to zero submissions.
